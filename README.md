# ProyectoSoftware_IS
TaskFlow: “Organiza tu trabajo, visualiza tu progreso.” 
Gestor web inteligente de tareas y flujo de trabajo para equipos pequeños.
---

# FUNCIONALIDADES PRINCIPALES
---
- Gestión de usuarios y equipos. (base de datos con json)
- Creación de proyectos y tareas.
- Asignación de responsables y fechas límite y duración para cada proyecto.
- Cambio de estado (Pendiente, En progreso, Finalizado).
- Panel de métricas básicas (tareas completadas, retrasos, sprints, etc).

# MODULOS DEL SISTEMA
---
- Login y gestión de usuarios.
- Panel de proyectos.
- Gestor de tareas (CRUD).
- Panel de métricas y reportes.
- *Tablero Kanban.
